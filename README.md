# 🛡️ AI-Powered Intrusion Detection System (Work in Progress)

An end-to-end AI-powered Network Intrusion Detection platform built with a microservices architecture. The system simulates network traffic, analyzes it with machine learning models, generates alerts, and displays them on a SOC dashboard.

> ⚠️ **This project is currently under active development — not yet complete.**

---

## 🏗️ Architecture

```
Traffic Simulator → Redis → ML Service → FastAPI Alert API → SOC Dashboard (Blazor)
                                                    ↓
                                             SOAR Engine (Automated Response)
```

---

## 🔬 ML Models

Training is based on the **NSL-KDD** dataset — a standard benchmark for network intrusion detection.

Models implemented so far :
- Random Forest
- SVM
- XGBoost
- Ensemble model

---

## 🛠️ Tech Stack

| Component | Technology |
|-----------|-----------|
| ML Service | Python, scikit-learn, XGBoost |
| Alert API | FastAPI, PostgreSQL |
| Streaming | Redis |
| Dashboard | .NET 8 Blazor Server |
| Containerization | Docker / Docker Compose |

---

## 📁 Project Structure

```
cloud-ids-soar/
├── streaming/          # Traffic simulator & Redis ingestion
├── ml_service/         # ML models, feature engineering, inference
├── api/                # FastAPI alert storage & WebSocket
├── alert_engine/       # Alert consumer & processor
├── soar/               # Automated response engine
├── Dashboard/          # Blazor SOC dashboard
└── docker-compose.yml  # Full stack orchestration
```
